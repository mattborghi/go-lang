# Cards

Run by running

```
make run
```

and

```
# go run 02_cards/main.go 02_cards/deck.go
```

or simply

```
# docker exec -it learning_go go run 02_cards/main.go 02_cards/deck.go
```

## Test

After `make run` we can do

```
# go test ./02_cards
```

or simply

or simply

```
# docker exec -it learning_go go test ./02_cards
```
